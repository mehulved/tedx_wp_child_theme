Tedx wp child theme

Child Theme based on WP Foundation by 320press, which is modified to add tedx flavour so that it can be used as a base for creating tedx websites.

View the Project on GitHub mehulved/tedx_wp_child_theme

This is a child theme of wp-foundation by 320press. It can be availed from https://github.com/320press/wordpress-foundation. This theme won't work without wp-foundation, so get that before activating this theme.

To display list of speakers, create a page for speakers, select template Speakers from the edit page view. Then add speakers from the Speakers menu. Follow the same steps for team.
